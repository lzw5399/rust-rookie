# rust-rookie

## Project setup
```
cargo build
```

### Run server
```
cargo run
```

### Compiles and minifies for production
```
cargo build --release
```

### Check compile result
```
cargo check
```

### Update dependency
```
cargo update
```

### Run your tests
```
cargo test
```
